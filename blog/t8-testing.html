What is scope and how does it work in JavaScript?

In JavaScript, scope is the set of variables that you have access to. JavaScript has two scopes â€“ local and global. A variable that is declared outside a function definition is a global variable, and its value is accessible and modifiable throughout a program. A variable that is declared inside a function is local and is only accessible within that function. A local variable is created and destroyed every time the function is executed. Local and global variables can have the same names, though they are entirely separate variables.

Local Scope Example

Consider the function adder() below. Adder declares and initialises the variables a = 1 and b = 2, which are both local to adder().Outside of the function, variable c is declared and initialised by calling adder(), which returns the value 3. You may expect that c is evaluated to 4 because a is set to 2 above the function call. However, the local variables a and b cannot be effected outside of the function. The a variable outside of the function is a different variable and it has global scope. 


						function adder() {

						 var a = 1;

						 var b = 2;

						 return a + b;

						}

						a = 2;

						var c = adder(); // c = 3;
Global Scope Example

Consider the function subtractor below. Global variable a is declared and initialised outside of the function. The function can access a's value. Local variable b is declared and initialised inside the function, and the function returns the value of global variable a minus the value of local variable b. 


						var a = 3;

						function subtractor() {

						 var b = 2;

						 return a - b;

						}

						var c = subtractor(); // c = 1;
