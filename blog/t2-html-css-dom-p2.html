What is grid-based design?

Grid-based design is a concept in graphic design in which a document is built on a grid structure of intersecting horizontal and vertical lines. Such grids are used to structure content on the document and can help with design aspects such as aligning content.

What's all the hype about responsive webpages?

Responsive websites are websites that are optimised to give a good user experience regardless of the type of device it is viewed on. People view website on a range of devices with a wide range of screen sizes including desktop computer, laptops, tablets and mobile phones. A responsive website will render itself according to the type of device (or screen size) it is being viewed on.

For example, a news website viewed on a monitor may include a header, footer, sidebar and content section. This layout may not be appropriate for a much smaller mobile phone, and so the sidebar could be dropped or put in a menu. CSS provides screen size querying functions that can be used to determine how a page is displayed based on the screen size in use.

What is semantic structure?

In HTML, semantic structure relates to using semantic elements (discussed below) to more rigidly define the structure and layout of a website. Semantic elements give some sort of description of the content of that element (e.g, table), whereas non-semantic elements give no hint as to content (e.g, div).

HTML provides a number of semantic elements including form, table and img. Further, HTML 5 introduced a number of new semantic elements including section, article, header, footer, aside and nav. These elements can be used by a web developer to more clearly lay out the structure of a website.
