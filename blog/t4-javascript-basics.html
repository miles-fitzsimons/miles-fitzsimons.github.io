How does JavaScript compare to HTML and CSS?

HTML is used to create the basic structure and actual content of a website. CSS is used for the styling the HTML elements of the page. CSS defines how how content is laid out and how it looks. Javascript is used for changing or animating the HTML elements once a website has loaded. For example, a button that when clicked makes text disappear can be built with JavaScript. JavaScript is an object-orientated programming language.

What is control flow and loops?

Control flow is the order in which individual statements, instructions or function calls of a program are executed or evaluated. Control flow can be executed by using one or more of a number of control flow constructs. For example, an if statement evaluates whether a condition is true and, if so, executes a code snippet. A for or while loop can be used to execute a code snippet a number of times to manipulate variables.

A number of everyday life examples of control flow are provided in psuedocode below.

Weather:
if ( it is cold && it is raining )
    { wear your coat and take your umbrella }
else if ( it is cold )
    { wear your coat }
else if ( it is raining )
    { take your umbrella }
else
    { leave now }

Bread:
while ( the dough is smaller than twice its original size )
    { let the dough rise in a warm place }

Coffee:
for ( ten times ) { stir }

What is the difference between accessing data from arrays and object literals?

An array type is a data type that describes a collection of elements (values or variables), each of which can selected by using one or more indices. For example, an array may describe a number of integers, such as: myArray = [2,5,1,0,4,7,3].

myArray has a length of 7. In JavaScript, the array index begins at zero. myArray[0] can be used to access the first integer (i.e, 2).

An object is a data structure that defines a number of key-value pairs and can be used to describe real world environments or scenarios. A value is accessed by calling the corresponding key. In contrast to arrays, object values are not indexed. For example, you might describe a car using object-literal notation, such as:
myCar = {
    colour: "black",
    wheels: 4,
    make: "Toyota",
    model: "Corolla",
    year: 2016
};

You could access the year value of the car by calling myCar[“year”] (bracket notation) or myCar.year (dot notation).

What are functions and why are they useful?

A JavaScript function is a block of code designed to perform a particular task. AJavaScript function is executed when something invokes it (calls it). Functions are useful in cases where the same block of code needs to be executed numerous times. This invokes the “DRY” (don’t repeat yourself) principle.
